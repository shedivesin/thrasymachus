QUICK GUIDE TO THE KEYBOARD

FOR THE SPIonic FONT*

GREEK CHARACTERS

 a = A  z = Z  l = L  p = P  u = U
 b = B  h = H  m = M  r = R  f = F
 g = G  q = Q  n = N s = S  x = X
 d = D  i = I  c = C  j = J  y = Y
 e = E  k = K  o = O t = T  w = W

 

DIACRITICAL MARKS

Press the key for the diacritical mark after you type the vowel to be marked.

Acute Accent = /

Grave Accent = \

Circumflex = ~

Rough Breathing = (

Smooth Breathing = )

Iota Subscript = Shift \

 

*For a complete listing of all the characters in SPIonic, see the Description at ftp://rosetta.atla-certr.org/pub/fonts/windows/SPIonic.readme
